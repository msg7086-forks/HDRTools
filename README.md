# HDRTools (Linux)

Avisynth HDR Tools plugin by jpsdr.

Ported to Linux64. Need cmake 3.22 (or compile it by hand). No multi-threading.

## To compile

```bash
cd HDRTools
mkdir build
cmake -B build -S .
cmake --build build
```
